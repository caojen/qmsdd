# QMSDD

+ C++ Implementation of QMSDD

## build
Use ``make`` to build the program ``./bin/main``

## usage

```bash
# after building
./bin/main <filename>
```

filename format:
```
<n, an Integer: variable count>
<fn, an Integer: function count>
loop for fn lines:
  for each line, there are 2^n symbol, 0, 1, or other ascii
```
